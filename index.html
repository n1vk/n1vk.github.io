这是一篇测试性的文章

在这篇文章里，我将会尝试使用 Markdown 来写作。

网站存在的目的

1.记录想法

一个博客性质的网站，可以记录自己生活中的随意想法。

2.技术存档

将自己解决过的问题进行存档，方便以后查阅/分享给其他人。

Markdown 具有得天独厚的优势，可以在网页上写出大方又美观的文档。

可以在这里学习 Markdown 的更多用法。

总而言之，维护自己的博客是一件很有趣的事情 :) ！
